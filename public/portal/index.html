WhatsApp Check
Check which of your leads are on WhatsApp
How automatic checking works
  1. Click Export numbers to check → save wa-numbers.json and move it into the wa-checker folder.
  2. In that folder run npm install once, then npm start. Scan the QR with WhatsApp (Settings → Linked devices) — one time.
  3. It goes through each number one by one, waiting a few seconds between each (editable), and writes WhatsApp Available / Not Available for every one.
  4. Click Load results file here and pick wa-results.json (or wa-results.csv) — statuses fill in across the dashboard.
The checker uses your own WhatsApp account. Checking hundreds of numbers can get it temporarily limited — it runs slowly and in batches on purpose. Use at your own risk.

How it works
  1. Drop or pick one or more files. Excel, CSV, the Google Sheets HTML export, or JSON all work.
  2. Columns are matched automatically (name, phone, website, address, rating, category, etc.) — order doesn't matter.

  5. Imported leads are saved in this browser and merged into every view.

WhatsApp Checker automated

Auto-detect which numbers are on WhatsApp — no checking one by one. Runs locally using your own WhatsApp via the included wa-checker tool.

  1. Click Export numbers below → save wa-numbers.json.
  2. Move that file into the wa-checker folder, then run npm install once, and npm start.
  3. Scan the QR code with WhatsApp (Settings → Linked devices) — one time only.
  4. It checks every number and writes wa-results.json.
  5. Drag wa-results.json onto the drop zone above — statuses appear across the dashboard instantly.
Checking hundreds of numbers from a personal WhatsApp can get it temporarily limited. The tool goes slow by default and lets you check in batches — see its README.
